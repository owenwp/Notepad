Comic Book Reader 2010 Zombie Process Independant Development Project

iTunes Link

My most recent project, also independent. Allows users to view comic book archive files (cbr, cbz, rar, zip). Uses the new iPad document handling features to allow it to open the files from other apps including Mobile Safari, and to add files straight from itunes. Also includes the ability to organize comic books into collections by filtering their file names, and will generate cover thumbnails to present the user's collection visually.

The user interface is strongly influenced by Apple's iBooks app, with a scrolling bookshelf representing the user's library. Core Graphics is used to composite 3D rendered images and shadows that make up the parts of the shelves. The volumes within a collection are presented using a coverflow view, which I adapted and enhanced from an open source framework. Threading is used for loading images to minimize load times.

The comic view itself uses nested scroll views, the outermost one used to handle movement from page to page, and each page has its own inner scroll view for zooming and panning. This creates a seamless transition from one page to another by dragging past the edge of a page, even when zoomed in close. A tap gesture is used to show and hide of the toolbars and status bar, as well as page turning by tapping in the margins. Pages are pre-fetched from the archive and cached to eliminate any delay during page turns, and the user's reading history is retained.
