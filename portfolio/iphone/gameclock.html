GameClock Tournament 2009 Zombie Process Independent Development Project

iTunes Link

My first iPhone project. A clock suitable for timing various two player board games, such as Chess and Go. Includes precise implementations of the most commonly used timing methods. Distributed as a basic free version with only sudden death mode, with in-app purchase for the rest of the timing modes.

The primary implementation concern for this app was battery life, as it is meant to be used for hours at a time without interruption. It proved necessary to implement my own compositing code to draw the digits on the timer, as allowing UIKit to render the text each second was costly. Much of the code for this app is also used in Tengen Go.
